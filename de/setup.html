Setup & Voraussetzungen

Was Sie brauchen, um loszulegen

Bevor Sie mit den Übungen starten, sollten Sie Ihre Arbeitsumgebung einrichten. Keine Sorge – alles hier ist kostenlos und für Einsteiger geeignet. Diese Seite führt Sie Schritt für Schritt durch die Installation.

Schnellcheck: Sind Sie startklar?

Prüfen Sie diese Punkte. Wenn alle erfüllt sind, können Sie direkt mit den Übungen starten.

  • ☐ Python installiert Terminal öffnen, tippen: python --version
  • ☐ VS Code installiert Können Sie VS Code öffnen?
  • ☐ Python-Extension in VS Code Extensions-Tab → Suche "Python" → Installiert?
  • ☐ LLM-Zugang Haben Sie Zugang zu einem LLM Ihrer Wahl?
  • ☐ Erstes Script ausgeführt Erstellen Sie test.py mit print('Hallo!') und führen Sie es aus

1. Python installieren

Python

Die Programmiersprache für unsere Code-Beispiele

erforderlich

Python ist die Programmiersprache, die wir in diesem Kurs verwenden. Sie ist bekannt für ihre Lesbarkeit und wird häufig in der Wissenschaft eingesetzt.

→ Python herunterladen (python.org)

Wir empfehlen Python 3.10 oder neuer.

Wichtig bei der Installation (Windows):

Aktivieren Sie unbedingt die Option "Add Python to PATH" am Anfang der Installation!

2. Visual Studio Code einrichten

Visual Studio Code

Der Code-Editor zum Schreiben und Ausführen von Scripts

erforderlich

VS Code ist ein kostenloser, leichtgewichtiger Code-Editor von Microsoft. Er ist einfach zu bedienen und hat eine große Sammlung von Erweiterungen.

→ VS Code herunterladen (code.visualstudio.com)

Verfügbar für Windows, macOS und Linux.

Tutorials

Microsoft: Getting Started with Python in VS Code

Offizielles 10-Minuten-Tutorial von Microsoft

empfohlen
Corey Schafer: VS Code Python Setup (Windows)

Ausführliches Setup-Tutorial für Windows

VS Code: Quick Start Guide for Python

Offizielle Dokumentation von Microsoft

Empfohlene Erweiterungen

Nach der Installation von VS Code, installieren Sie diese Erweiterungen:

  • Python ms-python.python

    Grundlegende Python-Unterstützung (Syntax-Highlighting, Debugging)

  • Pylance ms-python.vscode-pylance

    Verbesserte Code-Vervollständigung und Fehlerprüfung

So installieren Sie Erweiterungen:

  1. Öffnen Sie VS Code
  2. Klicken Sie auf das Extensions-Symbol in der linken Leiste (oder Ctrl+Shift+X)
  3. Suchen Sie nach "Python"
  4. Klicken Sie auf "Install"

Hinweis:

Alle genannten Optionen funktionieren für die Übungen. Wählen Sie einen Anbieter, bei dem Sie sich wohl fühlen. Die Konzepte sind übertragbar – Sie können jederzeit wechseln.

Persönliche Empfehlung:

Aus meiner Erfahrung bietet die Bezahlversion von Claude (Claude Pro) die spannendste Lernerfahrung – besonders für längere Code-Erklärungen und iteratives Arbeiten. Wiewohl das natürlich ein proprietäres Modell ist, ist es für den Einstieg eine gute Wahl.

5. Pakete installieren (pip)

pip (Paketmanager)

Zum Installieren von Python-Bibliotheken

erforderlich

pip wird automatisch mit Python installiert. Es ist der Paketmanager, mit dem Sie zusätzliche Bibliotheken installieren.

Installation prüfen

Öffnen Sie ein Terminal und geben Sie ein:

Terminal 
pip --version

Häufig verwendete Pakete

Diese Pakete werden in den Übungen verwendet:

  • pandas pip install pandas

    Datenanalyse und CSV/Excel-Verarbeitung

  • Pillow pip install Pillow

    Bildverarbeitung

  • matplotlib pip install matplotlib

    Diagramme und Visualisierungen

Tipp:

Sie müssen nicht alle Pakete vorab installieren. In den Übungen wird angegeben, welche Pakete benötigt werden.

6. Erstes Script ausführen

Testen Sie Ihre Einrichtung mit einem einfachen Script:

  1. Öffnen Sie VS Code
  2. Erstellen Sie eine neue Datei

    File → New File oder Ctrl+N

  3. Fügen Sie diesen Code ein:
    test.py 
    # Mein erstes Python-Script
print("Hallo, AI Coding Literacy!")
print("Python funktioniert!")

# Eine einfache Berechnung
ergebnis = 2 + 2
print(f"2 + 2 = {ergebnis}")
  4. Speichern Sie die Datei

    File → Save As → Speichern als test.py

  5. Führen Sie das Script aus

    Klicken Sie auf den Play-Button oben rechts, oder:

    Terminal öffnen (Ctrl+`) und eingeben: python test.py

Erwartete Ausgabe: 
Hallo, AI Coding Literacy!
Python funktioniert!
2 + 2 = 4

Wenn Sie diese Ausgabe sehen, ist alles richtig eingerichtet!

Häufige Probleme

"python" wird nicht erkannt (Windows)

Python wurde nicht zum PATH hinzugefügt. Lösung: Python neu installieren und "Add Python to PATH" aktivieren.

"python3" statt "python" (macOS/Linux)

Auf manchen Systemen heißt der Befehl python3 statt python. Probieren Sie: python3 --version

VS Code findet Python nicht

Öffnen Sie die Command Palette (Ctrl+Shift+P) und suchen Sie "Python: Select Interpreter". Wählen Sie die installierte Python-Version aus.

pip-Befehl funktioniert nicht

Versuchen Sie python -m pip --version statt pip --version.
