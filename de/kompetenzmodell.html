Didaktischer Ansatz

Das Curriculum folgt einem informed vibe coding Ansatz: Experimentieren ist erwünscht, aber mit wachsendem Verständnis für das, was passiert.

„Vibe coding" – der intuitive, explorative Umgang mit Code – wird nicht ersetzt, sondern durch systematische Reflexion ergänzt.

Prinzipien

  • Lesen vor Schreiben: Code verstehen ist einfacher als Code schreiben
  • Domänenwissen als Stärke: Ihr Fachwissen macht Sie zum Experten für Ihre Probleme
  • Iteration statt Perfektion: Schrittweise Verbesserung durch Dialog
  • Grenzen erkennen: Wissen, wann professionelle Hilfe nötig ist
