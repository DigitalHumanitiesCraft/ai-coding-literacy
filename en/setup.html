Setup & Prerequisites

What you need to get started

Before starting the exercises, you should set up your working environment. Don't worry – everything here is free and suitable for beginners. This page will guide you through the installation step by step.

Quick Check: Are you ready?

Check these points. If all are met, you can start directly with the exercises.

  • ☐ Python installed Open terminal, type: python --version
  • ☐ VS Code installed Can you open VS Code?
  • ☐ Python extension in VS Code Extensions tab → Search "Python" → Installed?
  • ☐ LLM access Do you have access to an LLM of your choice?
  • ☐ First script executed Create test.py with print('Hello!') and run it

1. Install Python

Python

The programming language for our code examples

required

Python is the programming language we use in this course. It's known for its readability and is widely used in science.

→ Download Python (python.org)

We recommend Python 3.10 or newer.

Important during installation (Windows):

Make sure to enable the option "Add Python to PATH" at the beginning of installation!

2. Set up Visual Studio Code

Visual Studio Code

The code editor for writing and running scripts

required

VS Code is a free, lightweight code editor from Microsoft. It's easy to use and has a large collection of extensions.

→ Download VS Code (code.visualstudio.com)

Available for Windows, macOS and Linux.

Recommended Extensions

After installing VS Code, install these extensions:

  • Python ms-python.python

    Basic Python support (syntax highlighting, debugging)

  • Pylance ms-python.vscode-pylance

    Improved code completion and error checking

How to install extensions:

  1. Open VS Code
  2. Click the Extensions icon in the left sidebar (or Ctrl+Shift+X)
  3. Search for "Python"
  4. Click "Install"

6. Run Your First Script

Test your setup with a simple script:

  1. Open VS Code
  2. Create a new file

    File → New File or Ctrl+N

  3. Paste this code:
    test.py 
    # My first Python script
print("Hello, AI Coding Literacy!")
print("Python is working!")

# A simple calculation
result = 2 + 2
print(f"2 + 2 = {result}")
  4. Save the file

    File → Save As → Save as test.py

  5. Run the script

    Click the Play button in the top right, or:

    Open terminal (Ctrl+`) and enter: python test.py

Expected output: 
Hello, AI Coding Literacy!
Python is working!
2 + 2 = 4

If you see this output, everything is set up correctly!

Common Problems

"python" is not recognized (Windows)

Python was not added to PATH. Solution: Reinstall Python and enable "Add Python to PATH".

"python3" instead of "python" (macOS/Linux)

On some systems the command is python3 instead of python. Try: python3 --version

VS Code can't find Python

Open the Command Palette (Ctrl+Shift+P) and search for "Python: Select Interpreter". Select the installed Python version.

pip command doesn't work

Try python -m pip --version instead of pip --version.
